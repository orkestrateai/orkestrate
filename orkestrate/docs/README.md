# CLI docs (repo contributors)

**Public documentation** for [orkestrate.space](https://orkestrate.space/docs) lives in the monorepo at [`website/src/app/docs/`](../../website/src/app/docs/). That is the canonical reference for install, registry, publisher flow, and the workbench.
